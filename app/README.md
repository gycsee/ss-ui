# Demo Apps

This directory contains runnable demos for validating the workspace packages:

- `react-demo`: demo for `@cardinal-odp/algo-params-react`
- `vue-demo`: demo for `@cardinal-odp/algo-params-vue`

Common commands from the repo root:

```bash
pnpm install
pnpm demo:react
pnpm demo:vue
pnpm demo:build
```
